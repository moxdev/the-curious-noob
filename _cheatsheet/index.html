Ruby Basics Cheatsheet

    IRB
  • type 'irb' to start irb
  • Exit IRB: `ctrl + D`

    Ruby Methods
  • .reverse - reverses string
  • .length - counts number of characters including spaces
  • .upcase - changes string to uppercase
  • .downcase - changes string to lowercase
  • .capitalize - capitalizes first character of string
  • .swapcase - switches the case of every letter in string
  • .center - centers the string
  • .push - adds an object to end of array
  • .pop - removes the last object from array (and tells you what it was)
  • .last - tells you whats at end of array
  • .join(), .join(" ") -will join and add a space between words
    Misc
  • puts "\n" to print an empty line between output
  • Chain methods: `object.reverse.upcase.lowercase`
  • String Interpolation: `#{variable}`
  • If/Elsif/Else structure: if/elsif/end
  • Unless structure: unless/else/end
  • New Array: array = []
  • New Hash: hash = {}
  • Chomp: `chomp` - deletes input
